Порядок обновления

Прямое обновление на GitHub Pages

  1. Открой файл data/cases.csv в Excel или прямо в GitHub.
  2. Обнови или добавь строки кейсов.
  3. Для нескольких значений в одной ячейке используй разделитель |.
  4. Сохрани файл как CSV UTF-8 с тем же именем или закоммить изменение через GitHub web editor.
  5. GitHub Pages опубликует обновление после коммита в репозиторий.
  6. Проверь пользовательскую витрину после публикации.
